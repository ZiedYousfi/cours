## Cybersec

Croissance des cyberattaques

- Phishing
- Ransomware

Types :

- Black hat : hacker malveillant
- White hat : hacker éthique
- Grey hat : entre les deux/doute

## Kali Linux

- Distribution Linux orientée sécurité informatique
- Outils intégrés pour tests d'intrusion, analyse de vulnérabilités, forensic, etc.

## Ingénierie sociale

- Manipulation psychologique pour obtenir des informations sensibles

Exemples :

Mail frauduleux, appels téléphoniques, faux sites web d'une vraise entreprise/organisation.

- Techniques : phishing, pretexting, baiting, quid pro quo, tailgating

## Vente de données

- Données personnelles collectées par les entreprises
- Entreprises hackées pour voler des données
- Vente sur le dark web
