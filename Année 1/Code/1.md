## HTML : HyperText Markup Language

Langage de balisage utilisé pour structurer et présenter le contenu sur le web.

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    <h1 class="ý">Hello, World!</h1>
</body>
</html>
```

Langage de programmation qui permet d'implémenter des fonctionnalités complexes sur les pages web.

```javascript
// Exemple simple : Afficher une alerte lorsque l'utilisateur clique sur un bouton

// Créer un bouton (nouvelle constante)
const button = document.createElement('button');

// Définir le texte du bouton (propriété textContent)
button.textContent = 'Clique moi dessus ';

// Définir une classe CSS pour le bouton (propriété className)
button.className = "my-button";

// Ajouter un gestionnaire d'événements pour le clic (callback function sur l'événement 'click')
button.addEventListener('click', () => {
    alert('Hello JavaScript!');
});

// Ajouter le bouton au corps du document (méthode appendChild)
document.body.appendChild(button);
```
