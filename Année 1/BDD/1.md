# Introduction à la base de données SQL

## Structural Query Language (SQL)

SQL (Structured Query Language) est un langage de programmation utilisé pour gérer et manipuler des bases de données relationnelles. Il permet de créer, lire, mettre à jour et supprimer des données dans une base de données.

### Bloc 1: le LDD (Langage de Déclaration de Données)

Le LDD est utilisé pour définir la structure de la base de données, y compris la création de tables et la définition des relations entre elles. Voici un exemple de commandes LDD pour créer une base de données simple pour une application de location de véhicules :

```sql
CREATE DATABASE loc_vehicules;

USE loc_vehicules;

CREATE TABLE vehicules (
    id INT AUTO_INCREMENT PRIMARY KEY,
    nom VARCHAR(50) NOT NULL,
    louee BOOLEAN DEFAULT FALSE
);

CREATE TABLE clients (
    id INT AUTO_INCREMENT PRIMARY KEY,
    nom VARCHAR(50) NOT NULL,
    prenom VARCHAR(50) NOT NULL
);

ALTER TABLE clients ADD COLUMN email VARCHAR(100);

CREATE TABLE locations (
    id INT AUTO_INCREMENT PRIMARY KEY,
    vehicule_id INT,
    client_id INT,
    date_debut DATE NOT NULL,
    date_fin DATE,
    FOREIGN KEY (vehicule_id) REFERENCES vehicules(id),
    FOREIGN KEY (client_id) REFERENCES clients(id)
);
```

### Bloc 2: le LMD (Langage de Manipulation de Données)

Le LMD permet de manipuler les données contenues dans les tables créées avec le LDD. Voici quelques exemples de commandes LMD :

```sql
-- Insertion de données
INSERT INTO vehicules (nom, louee) VALUES ('Renault Clio', FALSE);
INSERT INTO clients (nom, prenom, email) VALUES ('Dupont', 'Jean', 'jean.dupont@example.com');

-- Mise à jour de données
UPDATE vehicules SET louee = TRUE WHERE id = 1;

-- Suppression de données
DELETE FROM clients WHERE id = 1;
```

### Bloc 3: Le LID (Langage d'Interrogation de Données)

Le LID est utilisé pour interroger la base de données et récupérer des informations spécifiques. Voici quelques exemples de requêtes LID :

```sql
-- Sélection de tous les véhicules disponibles
SELECT * FROM vehicules WHERE louee = FALSE;

-- Sélection des clients ayant loué un véhicule
SELECT c.nom, c.prenom, v.nom AS vehicule
FROM clients c
JOIN locations l ON c.id = l.client_id
JOIN vehicules v ON l.vehicule_id = v.id;

-- Comptage du nombre de véhicules loués
SELECT COUNT(*) AS nombre_vehicules_loues FROM vehicules WHERE louee = TRUE;
```

### Bloc 4: Le LCID (Langage de Contrôle d'Intégrité de Données)

Le LCID est utilisé pour définir des règles d'intégrité et de sécurité dans la base de données. Voici quelques exemples de commandes LCID :

```sql
CREATE USER 'utilisateur'@'localhost' IDENTIFIED BY 'motdepasse';
GRANT SELECT, INSERT, UPDATE ON loc_vehicules.* TO 'utilisateur'@'localhost';
REVOKE DELETE ON loc_vehicules.* FROM 'utilisateur'@'localhost;
```

## SGBDR

Un SGBDR (Système de Gestion de Base de Données Relationnelle) est un logiciel qui permet de créer, gérer et manipuler des bases de données relationnelles. Parmi les SGBDR les plus populaires, on trouve MySQL, PostgreSQL, Oracle Database et Microsoft SQL Server.
